# Venus Birthday Website Spec --- V1

## Concept

Interactive cinematic birthday website combining space, literature, and
dark‑academia design.

## Stack

Next.js 14, TypeScript, React Three Fiber, Three.js, GSAP, Framer
Motion, TailwindCSS, Lenis.

## Story Flow

1.  Space intro
2.  Planet Venus reveal
3.  Shooting star travels to Earth
4.  Landing in a Dostoevsky-style library
5.  Interactive sections (bookshelf navigation)
6.  Letter, poems, gifts
7.  Final birthday message

## Sections

-   Hero
-   Meaning of Venus
-   Dostoevsky Room
-   Garden of Venus
-   Timeline
-   Letter
-   Poetry
-   Gift Boxes
-   Final Scene
